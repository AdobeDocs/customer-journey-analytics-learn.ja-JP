# データビューのコンポーネントタイプ設定

Customer Journey Analyticsのデータビューでコンポーネントを設定する際に、フィールドを指標とディメンションのどちらにするかを選択できます。 多くの場合、文字列はディメンションで、数値は指標である必要があります。 ただし、切り替えるのに適した使用例がいくつかあります。 見て！
