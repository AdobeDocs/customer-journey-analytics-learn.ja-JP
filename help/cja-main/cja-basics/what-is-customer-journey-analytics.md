# Customer Journey Analytics とは

Customer Journey Analytics（CJA）を使用すると、Adobe Experience Platform で利用可能な任意のデータを使用して、チャネル全体で顧客のジャーニーを分析し、Analysis Workspace の機能を活用して迅速なインサイトを得ることができます。
