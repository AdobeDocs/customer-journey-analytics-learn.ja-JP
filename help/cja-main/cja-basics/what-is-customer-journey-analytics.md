# Customer Journey Analyticsとは

Customer Journey Analytics(CJA) を使用すると、Adobe Experience Platformで利用可能な任意のデータを使用して、Analysis Workspaceの機能を活用し、迅速なインサイトを得ることで、チャネルをまたいで顧客のジャーニーを分析できます。
