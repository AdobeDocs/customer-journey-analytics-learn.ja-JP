# クイックセグメントの作成

[!DNL Customer Journey Analytics] プロジェクトで[!UICONTROL &#x200B; クイックセグメント &#x200B;]を直接作成し、完全な[!UICONTROL &#x200B; セグメントビルダー]の複雑さを回避します。 この機能を使用すると、簡単な基準に基づいてコンポーネントデータをセグメント化し、結果をすばやく確認できます。 詳しくは、[ドキュメント](https://experienceleague.adobe.com/ja/docs/analytics-platform/using/cja-components/cja-segments/quick-filters){target="_blank"}を参照してください。

>[!NOTE]
>
> Customer Journey Analytics インターフェイスで「[!UICONTROL &#x200B; フィルター]」の名前が「[!UICONTROL &#x200B; セグメント &#x200B;]」に変更されました。
