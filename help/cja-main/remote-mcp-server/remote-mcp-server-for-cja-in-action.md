# [!DNL Customer Journey Analytics]のリモート MCP サーバーの動作

Claude、ChatGPT、CursorなどのAI ツールをCJA（Adobe Customer Journey Analytics） MCP サーバーに接続して、データビューを検索し、レポートを実行し、自然言語を使用してエグゼクティブ対応のKPI インサイトを生成する方法を説明します。 このデモでは、リモート MCP サーバーが、Adobe Experience Platformでスケーラブルで管理された分析ワークフローと自動レポートをどのように実現するかを紹介します。

詳しくは、Content Analytics [ドキュメント](https://developer.adobe.com/analytics-mcp/docs/cja/){target="_blank"}を参照してください。
