# データビューのコンポーネントタイプ設定

Customer Journey Analytics のデータビューでコンポーネントを設定する際に、フィールドを指標とディメンションのどちらにするかを選択できます。多くの場合、文字列はディメンション、数値は指標にする必要があります。ただし、切り替えたほうがよいユースケースがいくつかあります。ご覧ください。
