# タイプキャスティングの使用

Customer Journey Analyticsの派生フィールドでタイプキャストを確認します。 タイプキャストは、高度な分析のために文字列を数値や日時に変換するなど、スキーマフィールドのデータタイプを調整します。 フィールドを簡単に設定したり組み合わせたりして、ビジネスの強力なインサイトを引き出します。

詳しくは、[ドキュメント](https://experienceleague.adobe.com/ja/docs/analytics-platform/using/cja-dataviews/derived-fields){target="_blank"}を参照してください。
