# 日付計算を使用して、イベント間の時間を計算します

Customer Journey Analytics派生フィールドのマスター日付計算。 Date Mathは、より深いユーザー行動分析のために、検索時間から注文時間などのイベント間の時間差を計算します。 ダイナミックな日付範囲と柔軟な設定を活用して、旅行&amp;観光業界などの分析を最適化。

詳しくは、[ドキュメント](https://experienceleague.adobe.com/ja/docs/analytics-platform/using/cja-dataviews/derived-fields){target="_blank"}を参照してください。
