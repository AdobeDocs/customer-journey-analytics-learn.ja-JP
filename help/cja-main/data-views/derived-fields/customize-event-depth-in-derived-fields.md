# イベントの深度のカスタマイズ

Customer Journey Analyticsの派生フィールドでイベントの深さを設定して、ページの深さ、内部検索の深さなどの分析をカスタマイズする方法を説明します。 この機能により、ユーザーはユーザーの行動を正確に追跡し、データ主導のインサイトを強化できます。

詳しくは、[ドキュメント](https://experienceleague.adobe.com/ja/docs/analytics-platform/using/cja-dataviews/derived-fields){target="_blank"}を参照してください。
