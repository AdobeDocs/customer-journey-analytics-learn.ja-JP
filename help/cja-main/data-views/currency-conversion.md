# Customer Journey Analyticsでの通貨換算

データビューの設定を使用して、customer journey analytics で通貨換算を実行します。 この機能を使用すると、異なる通貨で表示する複数の指標を作成できます。 詳しいドキュメントについては、[ 形式コンポーネント設定 ](https://experienceleague.adobe.com/docs/analytics-platform/using/cja-dataviews/component-settings/format.html?lang=ja#currency){target="_blank"} を参照してください。
