# Analysis Workspaceでの動的ドロップダウンフィルターの作成

ディメンション全体に基づくドロップダウンフィルターを作成する方法を説明します。 Dimension項目は、レポートパネルで使用される日付範囲と、同じパネルの他のドロップダウンフィルターで選択された値に基づいて、動的に更新されます。
