# 注釈を作成

キャンペーンの開始、データの問題、休日などのイベントが発生した場合に、[!DNL Customer Journey Analytics] プロジェクトで[!UICONTROL 注釈]を作成する方法を説明します。この機能は、これらの日付または日付範囲における指標の相違をユーザーに通知します。
