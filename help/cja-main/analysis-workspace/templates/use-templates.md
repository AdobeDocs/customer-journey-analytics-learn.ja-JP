# テンプレートの使用

[!DNL Customer Journey Analytics]のテンプレートを使用して、分析とレポートを効率化する方法を説明します。 テンプレートは、web サイトの訪問、検索キーワード、訪問者の期間など、一般的なビジネス分析用に事前に構築された設定です。 このビデオでは、Analysis Workspaceでテンプレートを選択、プレビュー、カスタマイズして、カスタマイズされたプロジェクトを作成する方法を説明します。

詳しくは、[ドキュメント](https://experienceleague.adobe.com/ja/docs/analytics-platform/using/cja-workspace/templates/use-templates){target="_blank"}を参照してください。
