# マップのビジュアライゼーションの設定と使用

マップビジュアライゼーションを活用して位置データを視覚化しましょう。 選択ツールを使用して、ジオクラスターを特定し、その場でセグメントを作成します。 簡単な設定手順を少し説明するだけで、準備完了です。 詳しくは、[ドキュメント](https://experienceleague.adobe.com/ja/docs/analytics-platform/using/cja-workspace/visualizations/map){target="_blank"}を参照してください。
