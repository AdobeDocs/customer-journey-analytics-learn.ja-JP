# Analysis Workspaceでのコンポーネントドロップダウンの使用

コンポーネントドロップダウンを使用して、互換性のあるコンポーネントのリストから選択し、ビジュアライゼーションを構築します。 必要なものが見つからない場合は、ドロップダウンから直接新しいコンポーネントを作成することもできます。 これらのスペースは、より高度なユースケースの場合には、ドロップゾーンとしても引き続き機能します。
