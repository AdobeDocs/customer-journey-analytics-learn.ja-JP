# [!DNL Customer Journey Analytics]の[!DNL Analysis Workspace] プロジェクトに[!UICONTROL 行]のビジュアライゼーションを追加

[!DNL Customer Journey Analytics]の[!DNL Analysis Workspace] プロジェクトで[!UICONTROL 行] ビジュアライゼーションを追加および設定する方法について説明します。 このビジュアライゼーションは、値が時間の経過とともにどのように変化するかを示すために、行を使用して指標を表します。

>[!NOTE]
>
>[!DNL Analysis Workspace]のユーザーインターフェイスと機能は、[!DNL Adobe Analytics]と[!DNL Customer Journey Analytics]で似ています。
