# Customer Journey Analyticsでの線のビジュアライゼーション

折れ線グラフのビジュアライゼーションで時間の経過に伴う指標の理解を深める方法を説明します。 Customer Journey Analyticsを使用すると、X 軸/Y 軸をカスタマイズし、最小/最大ラベルを表示し、トレンドラインを追加してデータインサイトを向上させることができます。
