# 日付フィールドのサポートの活用

日付フィールドのサポートにより、Customer Journey Analytics 内の日付および日時フィールドの連携機能が追加されます。日付または日時フィールドがどのように表示されるかを選択したり、含む／除外機能で日付専用の論理演算子を使用して表示内容をフィルタリングしたりできます。
